## 폰트 로딩 종류

### FOUT [ 파웃 ]

Flash of Unstyled Text

기본 폰트로 먼저 화면 표시 후 다운로드한 폰트로 대체

a: 기존폰트(f)..아웃(out..)

- edge브라우저

 

### FOIT [포잇]

Flash of Invisible Text

폰트 다운로드가 끝날 때까지 화면에 텍스트 표시하지 않음

a: 폰트 기다려잇

- 브라우저 : Chrome, 사파리, 파이어폭스 등
- 크롬 FOIT은 폰트 다운로드를 3초만 기다리고, 그 이상 지연되면 대체 폰트로 표시 후 폰트 대체
- 

### FOFT [포프트]

Flash Of Faux Text

## 폰트 최적화 방법

```jsx
@font-face {
	font-family : 폰트명;
	src : url('data:font/woff2;charset=utf-8;base64,d09....AAA=') format('woff2'),
				url('./assets/fonts/subset.woff') format('woff'),
				url('./assets/fonts/normal.ttf') format('truetype');
	font-display : block;
}
```

1. 폰트 적용 시점 제어
    - `@font-face` - `font-display`
    
    ```jsx
    @font-face {
    	font-family : ...;
    	src : url('./assets/fonts/zenna.ttf');
    	**font-display : fallback;**
    }
    ```
    
    1. auto 
    2. block : FOIT(timeout = 3s)
    3. swap : FOUT
    4. fallback : FOIT ( timeout = 0.1s ), 3초 후에도 불러오지 못한 경우 기본 폰트로 유지, 이후 캐시하여 다운로드한 폰트로 바로 표시
    5. optional : FOIT ( timeout = 0.1s ), 이후 네트워크 상태에 따라 기본 폰트로 유지할지 결정. 이후 캐시
2. 폰트 사이즈 축소(Transfer.org)
    - 압축률 좋은 폰트 포맷으로 변경하기
        - 일반적으로 사용하는 TTF는 크기가 큼 → WOFF로 대체 (일부 브라우저에서 미지원..대체로는 지원)
    - 서브셋 폰트 : 필요한 문자의 폰트만 다운로드하기
        - transfer.org에서 원하는 글자만 옵션으로 넣고 해당 서브셋 폰트를 다운로드 가능
    - 파일이 아닌 Data-Url 형태의 폰트 사용하기
        - Data-URI : data스킴이 접두어로 붙은 문자열 형태의 데이터
            - 파일을 문자열 형태로 변환하여 문서에 인라인으로 삽입
        - 별도의 네트워크 로드 없이 App.css에서 폰트 사용 가능 ( 대신 App.css 로딩이 오래걸릴 수도)
